# SocialNodeJs
This is a backend application for a social media application written in node.js


## Run Code
<pre><code>
- make sure .env file is included - 
npm install
npm start
</code></pre>


## Test Code
<pre><code>
- make sure .env file is included - 
npm install
npm test
</code></pre>
